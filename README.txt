Dataseed Visualisation
---------------

In order for this module to work, you will first need a valid Dataseed account, and the dataseed Drupal module installed.


  - Go to the dataseed module config page (admin/config/system/dataseed) and enter your
  Dataseed credentials

  - the drupal path visualisation/<dataset_id>/<visualisation_id> will show the
  embedded visualisation whose id (on Dataseed) is <visualisation_id> for the
  dataset <dataset_id>
